# Celery 课程开发平台

**学习分布式任务处理 Celery 的综合教育平台**

---

## 概述

本仓库包含一个完整的教育平台，用于学习 Celery——Python 的分布式任务队列库。课程设计旨在将学生从初学者水平培养到专家水平，提供多个专业方向路径和全面的动手学习体验。

## 课程结构

### 学习路径

1. **基础路径**（8周）
   - 分布式任务处理入门
   - Celery 基础应用和配置
   - 任务定义、路由和监控
   - 安全基础知识

2. **专业开发**（6周）
   - 高级任务工作流和画布操作
   - 性能优化和监控
   - 错误处理和调试
   - 生产环境部署策略

3. **专家开发**（8周）
   - 企业级架构和模式
   - 可扩展性和高可用性
   - 高级安全和合规性
   - 自定义扩展和集成

4. **专业方向**（4-12周）
   - Django 集成专家
   - 性能优化专家
   - 架构和设计专家

## 快速开始

### 学生使用指南

1. **选择你的学习路径**
   - 初学者：从基础路径开始
   - 有经验的开发者：评估并加入适当级别
   - 专业方向：先完成基础和专业路径

2. **设置开发环境**
   ```bash
   # 克隆仓库
   git clone <repository-url>
   cd github_celery_course_dev

   # 启动开发环境
   docker-compose up -d

   # 访问课程材料
   cd course/
   ```

3. **开始学习**
   - 按照选择路径的模块顺序学习
   - 在继续之前完成所有评估
   - 参与交互式实验室和练习
   - 构建毕业项目

### 教师使用指南

1. **查看课程材料**
   - `COURSE_OUTLINE_CN.md` - 完整课程大纲
   - `LEARNING_PATH_GUIDE_CN.md` - 学习路径指南
   - `PRACTICAL_EXERCISES_CN.md` - 实践练习集合

2. **设置教学环境**
   ```bash
   # 部署教学环境
   docker-compose -f docker-compose.teaching.yml up -d

   # 访问监控仪表板
   # http://localhost:7001 (Flower)
   # http://localhost:3000 (自定义仪表板)
   ```

3. **自定义课程内容**
   - 根据需要修改模块规范
   - 添加机构特定示例
   - 自定义评估标准
   - 设置自定义交互组件

## 先修要求

### 技术要求
- **Python：** 版本 3.9 或更高
- **系统：** 最低 8GB RAM，推荐 16GB
- **存储：** Docker 和示例需要 20GB 可用空间
- **网络：** 文档和示例需要互联网连接

### 软件要求
- **Docker 和 Docker Compose：** 最新稳定版本
- **Git：** 代码示例的版本控制
- **IDE/编辑器：** Python 开发环境
- **终端/Shell：** 命令行界面

### 知识先修要求
- **Python 编程：** 需要扎实理解
- **基础 Linux：** 命令行操作
- **网络概念：** 基本理解有帮助
- **数据库知识：** SQL 基础有益
